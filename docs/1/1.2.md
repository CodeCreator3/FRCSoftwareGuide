# 1.2: Install VSCode
VSCode is an IDE (Integrated development environment)  
![VSCode logo](../images/1.2.1.png)  
It's like Microsoft Word, but for code  
VSCode is the only IDE supported by WPILib  
WPILib has a special version of VSCode  
- Includes extra tools for FRC teams  
<span style="color: green;">Download here (or search “Install WPILib”):</span> https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html  
![VSCode example](../images/1.2.2.png)
<span style="color: green;">Click on the “Extensions” button on the left side of the screen</span>  
![Extensions](../images/1.2.3.png)
<span style="color: green;">Install “Extension Pack for Java”</span>  
![Extension](../images/1.2.4.png)
This will help with testing Java code
You may need to reload window  
No need to install a JDK (already installed with WPILib VSCode)  
You can close out of the welcome screen
